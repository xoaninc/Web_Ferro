== Description ==

Tema WordPress para web de trenes y blog ferroviario. Mantiene la estructura estática original con integración completa de WordPress para posts, búsqueda y comentarios.

== Características ==

* Estructura estática preservada: Mantiene todo el diseño y funcionalidad original del HTML
* Posts dinámicos de WordPress: Los artículos se cargan desde WordPress
* Búsqueda híbrida: Combina búsqueda en WordPress y contenido estático
* Mapeo de categorías: Sistema de correspondencia entre categorías HTML y WordPress
* Sistema de comentarios: Compatible con comentarios de WordPress
* Responsive: Mantiene toda la responsividad original
* Calendario ferroviario: Sistema de eventos y calendario integrado
* Perfil de autor: Sistema completo de perfiles con redes sociales y Gravatar
* Filtros avanzados: Sistema de filtrado por categorías múltiples

== Instalación ==

1. Sube el tema a /wp-content/themes/ferrocarril-esp/
2. Activa el tema desde Apariencia > Temas
3. Las categorías se crearán automáticamente al activar el tema
4. Configura el nombre del blog en Ajustes > Generales
5. Añade una descripción del blog

== Mapeo de Categorías ==

El tema incluye un sistema de mapeo entre las categorías del HTML original y WordPress:

Categorías Históricas:
* ancho-iberico
* ancho-metrico
* ancho-internacional
* lineas-cerradas
* proyectos-cancelados
* proyectos-actuales
* proyectos-en-marcha

Categorías de Noticias:
* noticias, metro, tranvia, ave, cercanias

* evento-especial, mantenimiento, aniversario, cambio-horarios

Categorías de Ciudades:
* sevilla, madrid, barcelona, valencia, bilbao

== Frequently Asked Questions ==

= ¿Puedo usar este tema con el editor de bloques? =
Sí, el tema es compatible con el editor de bloques de WordPress y también incluye soporte para temas de bloques mediante theme.json.

= ¿Cómo configuro las redes sociales del autor? =
Ve a Usuarios > Tu Perfil y completa los campos de redes sociales en la sección "Información de Redes Sociales".

= ¿Las categorías se crean automáticamente? =
Sí, todas las categorías necesarias se crean automáticamente al activar el tema.
